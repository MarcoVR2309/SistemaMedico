Sistema Médico - Gestión de Citas (Proyecto Web)

Este es un proyecto de aplicación web desarrollado en ASP.NET y C# para la gestión de una clínica médica (Clínica Aguirre). El sistema permite a los pacientes iniciar sesión, ver su perfil, y gestionar la reserva de citas médicas, interactuando con una base de datos en Microsoft Azure.
